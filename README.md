<h2>💰 Controle Financeiro Martins 2.0</h2>

<p>
  Aplicação web para controle de finanças pessoais, com autenticação de usuários, registro de transações, geração de relatórios e exportação de dados. Desenvolvida com Python + Flask e hospedada na Vercel.
</p>

  🔗 <strong>Deploy:</strong> <a href="https://controle-financeiro-martins-2-0.vercel.app">controle-financeiro-martins-2-0.vercel.app</a>

## 📁 Estrutura do Projeto

```
Controle-financeiro-martins-2.0/
├── api/              # Endpoints da API
├── static/           # Arquivos estáticos (CSS, JS, imagens)
├── templates/        # Templates HTML (Jinja2)
├── app.py            # Aplicação principal Flask
├── auth.py           # Módulo de autenticação
├── database.py       # Configuração do banco de dados
├── export_utils.py   # Utilitários de exportação
├── repos.py          # Camada de repositório
├── requirements.txt  # Dependências do projeto
├── vercel.json       # Configuração de deploy na Vercel
└── runtime.txt       # Versão do Python
```

---

## 🚀 Como rodar localmente

```bash
# Clone o repositório
git clone https://github.com/carlosmartins-ti/Controle-financeiro-martins-2.0.git

# Acesse a pasta
cd Controle-financeiro-martins-2.0

# Instale as dependências
pip install -r requirements.txt

# Execute a aplicação
python app.py
```

Acesse em: `http://localhost:5000`
